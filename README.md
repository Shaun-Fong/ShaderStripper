# ShaderStripper
Personal Shader Stripper Tool For Unity.

个人Shader变体工具

在[UnityShaderStripper](https://github.com/SixWays/UnityShaderStripper)工具的基础上合并了几个实用的工具，并进行了一些BUG的修复

测试的Unity版本为6000.0.34，其它版本未进行测试。理论上来说2022+以上的都没啥问题。
